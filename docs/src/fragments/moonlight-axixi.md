# 关于「月光·阿西西」及其作者

> [!WARNING] 时效性提示
>
> 此片段编写于 2025-12-15，最后更新于 2025-12-28

「月光·阿西西」鸿蒙版本违反 GPL-3.0 协议闭源[^1]

`Axixi2233/moonlight-android` 项目早期未开源，且在开源后长期不同步仓库代码更新[^2]，装样子开源，同时违反了 GPL-3.0 和 OSD

同时，此作者也被指控剽窃代码、独揽功劳、态度不端、人品不正[^3]

这些丑闻我是最近才知道的，没想到你原来是这样的人么？软件做得挺好，人做得不行，比起闭源，我更反感<nt text="假开源" max-width="500px"><p>「以开源之名，行不义之事」</p><p>实际上所有宣称自己是开源软件但仅仅是半开源（仅开源部分代码，不符合 OSD）甚至没有开源的都能被称为假开源</p></nt>

[^1]: 「月光·阿西西」鸿蒙版的上游 [Moonlight-OHOS](https://gitee.com/smdsbz/moonlight-ohos) 使用了 Moonlight 的 GPL-3.0 组件却以 MIT 协议开源，所以实际上违反其的是 Moonlight-OHOS，但由于其传染性，「月光·阿西西」的鸿蒙版也间接违反了其

[^2]: [GitHub 仓库 250210 到 250916](https://github.com/Axixi2233/moonlight-android/compare/250210...250916) 里这段时间的仓库没有任何实质性更新，却发布多个 Releases
      [Release 2501218](https://github.com/Axixi2233/moonlight-android/releases/tag/251218) 声称已经完整开源

[^3]: 我是从在哔哩哔哩里刷到的的一条视频最初了解到了这些信息 后面又发现此作者还发布了多条类似视频
