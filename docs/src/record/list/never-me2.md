## 为什么不要「+1」

还记得正确方式里的「有进展时收到通知」吗？
没错，你的一个「+1」，会让所有关心这个 Issue / PR / Discussion 的人收到一封电子邮件，他们会以为事情有了进展，这里不是 QQ 等即时通信软件！

所以：仅当你有额外信息要补充时，才回复！
